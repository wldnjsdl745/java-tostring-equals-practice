# java-tostring-equals-practice
자바 `toString()`과 `equals()` 메서드 오버라이딩을 연습한 프로젝트

## 학습 내용
- `toString()` 오버라이딩
- `equals()` 오버라이딩
- 동일성(`==`)과 동등성(`equals()`) 비교

## 프로젝트 구성
- `Rectangle.java`
- `RectangleMain.java`

## 핵심 개념
- `==` : 객체의 참조값 비교
- `equals()` : 객체의 내용 비교
- `toString()` : 객체 정보를 문자열로 반환
